# 🤝 Spolupráce na NexusForge

Vítáme každou pomocnou ruku! Pokud chceš přispět, postupuj následovně:

1. **Forkni** tento repozitář.
2. Vytvoř si **vlastní větev** (`git checkout -b feature/uzasna-funkce`).
3. Proveď změny a **commitni** je (`git commit -m 'Přidána úžasná funkce'`).
4. **Pushni** větev na svůj fork.
5. Otevři **Pull Request** (PR).

### Co hledáme?
- Optimalizace komunikace s Ollamou.
- Nové typy agentů (např. agent pro analýzu souborů).
- Vylepšení stability Docker kontejnerů.
