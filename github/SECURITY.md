# 🔒 Bezpečnostní politika

## Podporované verze
V současné době aktivně podporujeme pouze verzi **Main**.

## Hlášení zranitelností
Prosíme, nepoužívejte GitHub Issues pro hlášení bezpečnostních chyb. Místo toho mi pošlete zprávu přímo skrze GitHub profil nebo na můj kontaktní email (pokud je uveden).

### Důležité upozornění
Tento projekt spouští kód uvnitř Docker kontejnerů. Přestože je to izolované prostředí, doporučujeme nespouštět agenty s právy `root` v produkčním prostředí.
